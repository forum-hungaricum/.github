| [`CODE_OF_CONDUCT.md`](./CODE_OF_CONDUCT.md) | Community guidelines (Contributor Covenant 2.1 + apolitical-by-genre addition) |
| [`CONTRIBUTING.md`](./CONTRIBUTING.md) | Contribution guidelines (under strategic-transparency framework) |
